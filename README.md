# SOAR
> Train. Evolve. Soar.
SOAR is a fitness RPG where real-world workouts shape a unique character through progression, discipline, and muscle-specific growth.

The goal is to make consistency feel rewarding by turning fitness into a long-term game rather than a streak counter.

Instead of a single XP bar, workouts contribute to muscle-specific growth, discipline, and character evolution, creating a unique avatar based on how each user trains.

The goal is to make fitness feel like progression in a game while still reflecting real-world effort.

## Why I Started This

Most fitness apps reward users with streaks and numbers.

I wanted to explore a different approach where every workout changes a character's physique and long-term consistency unlocks new arenas inspired by RPG progression systems.

SOAR is an experiment in combining game design with fitness to make progress feel visible, meaningful, and worth coming back for.
